## CAJA RAPIDA
### Description
Servicio de backend para punto de venta

### Requirements
Docker

### Run
* ```docker network create crnet```
* ```docker-compose build ```
* ```docker-compose up -d service```
* running on [http://localhost:9091/query](http://localhost:9091/query)

### GraphQL playground
